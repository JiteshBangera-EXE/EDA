#  Used Car Price Analysis — Exploratory Data Analysis (EDA)

##  Objective
The goal of this project is to explore a used car dataset and understand:
- What factors influence the selling price of cars
- How numerical and categorical features affect price
- Patterns, trends, and outliers in the data

This EDA will also prepare the dataset for building a **machine learning regression model** to predict car prices in the future.

##  Key Insights

- Newer cars generally have higher prices.
- Cars with higher kilometers driven tend to have lower prices.
- Transmission type affects price: Automatic cars are usually more expensive.
- Fuel type also influences pricing.
- Engine size and power show positive correlation with price.
- Some outliers exist in price and kilometers driven which should be handled before model training.
